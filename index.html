Privacy Policy for n8n Automate Pins

Last updated: February 2025

n8n Automate Pins ("the Service") is an automation workflow created using n8n to help users publish or schedule Pinterest Pins programmatically. This Privacy Policy explains how we handle data when you use our Service.

1. Information We Collect

The Service may collect or process Pinterest API data, workflow input data, and images needed to publish pins.

2. How We Use Your Data

Your data is only used to publish Pinterest Pins and manage automated workflows.

3. Data Storage

No permanent user data is stored. All data remains inside your self-hosted n8n instance.

4. Third-Party Services

Pinterest API is used to authenticate and publish content.

5. Security

You are responsible for securing your n8n environment. The Service itself never transfers data externally.
